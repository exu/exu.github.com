<f.st......
  |||||||||
  ||||||||+- x - extended attributes
  |||||||+- a - ACL info changed
  ||||||+- u - reserved for future
  |||||+- g - group
  ||||+- o - owner
  |||+- p - permissions
  ||+- t - modification time 
  |+- s - size
  +- c - checksum


Additional information:
. - no change
* - e.g. *deleting
c - local change/creation e.g. creating dir symlink
< - file transfer to remote
> - file transfer to local host (received)
