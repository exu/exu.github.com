--- layout: post title: VIM faster Text-blocks with onoremap date: 2012-06-18 16:00:01 UTC updated: 2012-06-18 16:00:01 UTC comments: false categories: text-block vim --- I've found nice VIM feature to add mapping to operators 

onoremap p i(
onoremap m i{
onoremap " i"
onoremap ' i'

Now i can press 
c' d'
to use defined text-blocks

This technique didn't work with visual mode (you must press vi')
